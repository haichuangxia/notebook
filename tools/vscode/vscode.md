| 类别       | 功能                       | 快捷键              |
| ---------- | -------------------------- | ------------------- |
| 行编辑     | 上/下移动当前行            | `alt+up/down`       |
|            | 向相邻行复制当前行         | `shift+alt+up/down` |
|            | 删除当前行                 | `ctrl+shift+k`      |
| 格式化     | 格式化整个文件             | `shift+alt+f`       |
|            | 格式化所选区域             | `ctrl+k,ctrl+f`     |
| 折叠代码   | 折叠选定代码               | `ctrl+shift+[`      |
|            | 展开选定代码               | `ctrl+shift+]`      |
|            | 折叠所有方法等             | `ctrl+k,ctrl+0`     |
|            | 规定折叠等级(如折叠到二级) | `ctrl+l,ctrl+[0-5]` |
| 编辑器视图 | 拆分编辑器                 | `ctrl+\`            |
