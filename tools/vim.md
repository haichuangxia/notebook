# vim插件管理：vim-plug

vim插件推荐：插件推荐使用vim-plug进行管理

| 插件         | 作用             |
| ------------ | ---------------- |
| nerdtree     | 显示文件数       |
| YoCompleteMe | 提供代码补全功能 |
| Solarized    | 主题方案         |
| utilsnips    | 代码补全         |
| vista        | 查看tag          |

vim基本配置与插件安装配置流程为：

1. 下载vim-plug
2. 修改`.vimrc`,  增加相关配置
3. 在vim中输入`:source ~/.vimrc`,使配置生效
4. 在vim中输入：`PlugInstall`, 下载插件
