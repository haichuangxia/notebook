# ip

## 特殊ip地址

### 127 VS localhost  VS  0.0.0.0

| 地址      | 特点                                                         |      |
| --------- | ------------------------------------------------------------ | ---- |
| 127.*     | 也叫做本地环回地址，所有发送本地环回地址的数据包都不会出现在本机以外的其他地方 |      |
| localhost | 一个域名，大多数localhost都指向了127.0.0.1这个IP地址         |      |
| 0.0.0.0   | 本网络中的本机，如果socket server的地址绑定的是0.0.0.0，则表示可以用本机的任何一个ip地址访问，无论是内网还是公网ip |      |
